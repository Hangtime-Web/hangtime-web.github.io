Finding Fun
Made Easy

Hangtime is an app that allows people to find nearby events in a simple, intuitive fashion. No more scrolling through groupchats, scraping facebook, or walking aimlessly around campus. With Hangtime, fun events are a simple click away.

Download now on the app store

Core Features

Crowdsourced events

Hangtime allows anyone to put their events on the app. Anything from a pickup basketball game to an impromptou concert can be put.

Map Interface

All Hangtime events can be viewed in a simple map interface.

Group chat for each event

Each event has a groupchat that you can join or leave at your will.

Featured Events

Don't trust the public? Hangtime also has featured events seperate from the crowdsourced ones.

Partnerships

Hangtime has partnerships with various fraternities, clubs, and resturants in the Austin area.

Free

All of this is completely free. All you have to do is download the app!
